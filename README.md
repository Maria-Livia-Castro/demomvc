# Gerenciador de Tarefas ✅

Aplicação web desenvolvida com Spring Boot para gerenciar tarefas com nome, responsável e data de entrega. Permite adicionar, editar, excluir e listar tarefas.

## 🛠 Tecnologias
- Java
- Spring Boot
- MySQL
- JPA / Hibernate

## 📌 Funcionalidades
- Adicionar nova tarefa
- Editar tarefa existente
- Excluir tarefa
- Listar todas as tarefas

## ⚙️ Requisitos
- Java 17+
- MySQL
- Maven

## 🚀 Como executar
1. Clone o repositório
2. Configure o `application.properties` com suas credenciais do MySQL
3. Execute o projeto com Maven ou sua IDE
